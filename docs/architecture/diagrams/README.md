# Architecture Diagrams

This folder contains architecture diagrams for AgenticLab.

## Formats

- **Mermaid** (`.mmd`) — rendered natively in GitHub and VS Code
- **PNG/SVG** — exported versions for presentations
- **Draw.io** (`.drawio`) — editable diagrams

## Diagrams

| File | Description |
|------|-------------|
| `agent-lifecycle.mmd` | Agent execution lifecycle |
| `local-architecture.mmd` | Local-only deployment |
| `hybrid-architecture.mmd` | Hybrid local + cloud deployment |

## Rendering

Mermaid diagrams can be previewed in VS Code with the **Mermaid** extension or rendered on GitHub automatically.
